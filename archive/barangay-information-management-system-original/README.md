# 🏛️ BIMS - Barangay Information Management System

**© 2025 Apphorialabs - Proprietary Software**

A comprehensive web-based system for managing barangay information, residents, households, and administrative tasks with advanced GIS mapping capabilities.

> **⚠️ PROPRIETARY SOFTWARE**: This project is owned by Apphorialabs and is for internal development use only. External contributions are not accepted.

## 🚀 Quick Start

```bash
# Clone and setup
git clone https://github.com/Apphorialabs/BIMS
cd bims
npm run setup
npm run dev
```

- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:5000

## 🏗️ Technology Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS, Leaflet.js
- **Backend**: Node.js, Express.js, PostgreSQL with PostGIS
- **Features**: JWT Auth, Redis Caching, GIS Mapping, Audit Logging

## 🎯 Key Features

- 👥 **Resident Management** with dynamic classifications
- 🏠 **Household Management** with family tracking
- 🗺️ **GIS Integration** with interactive maps
- 📋 **Administrative Tasks** with workflow processing
- 📊 **Reporting & Analytics** with comprehensive statistics
